# UE IPR Registry  
European Identity Primary Record Registry

Questo repository costituisce un **registro pubblico sperimentale**
di **Identity Primary Record (IPR)**.

Un IPR è un record di identità digitale progettato per attestare:
- l’esistenza temporale di un soggetto
- la titolarità di opere, sistemi o progetti
- la continuità verificabile nel tempo

Il registro è orientato a contesti:
- UE / istituzionali
- Horizon Europe
- ricerca, innovazione e governance digitale

Ogni IPR è documentato in modo:
- pubblico
- verificabile
- tracciabile
- auditabile
